# csp_PT

## Requirements for CSP Peformance Task
* List or Array: A data structure that stores multiple pieces of related information.
* Custom Procedure: A student-created function or method with one or more parameters.
* Algorithms: The procedure must contain an algorithm using sequencing (steps in order), selection (an if/else statement), and iteration (a for/while loop).
* Input and Output: The program must take user inputs and display outputs based on those inputs.

## My CSP Performance Task
* Idea: Song Shuffler
* Input as many songs as you want
* Input how long you want the list of outputted shuffled songs to be
* Function shuffles the songs
* Outputs the list of shuffled songs


## Requirements Fufilled
* List: Used to keep track of the songs inputted and the shuffled songs
* Function: Used to shuffle songs
* Sequencing: Used with lines of code in order
* Selection: Used with if/elif code to make sure same song doesn't come 2x consecutively
* Iteration: Used with while loop to keep adding songs to shuffled list until it reaches the amount specified
* Input: Used with asking player songs and lenght of shuffled list
* Output: Given as the list of shuffled songs
